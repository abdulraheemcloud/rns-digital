# Project Screenshots

Add your website screenshots here as .webp files at 1200x800 pixels.

Files needed:
- restaurant.webp  (Spice Garden Restaurant demo)
- dental.webp      (Smile Care Dental demo)
- realestate.webp  (Prestige Properties demo)
- gym.webp         (Fit Zone Gym demo)
- tuition.webp     (Bright Minds Tuition demo)
- boutique.webp    (Riya Boutique demo)

How to create:
1. Open demo site in Chrome
2. Press F12 → toggle device toolbar → set 1200x800
3. Screenshot → save as .webp (use squoosh.app if needed, quality 80)
4. Place in this folder

Until screenshots are added, cards show a gradient placeholder automatically.
